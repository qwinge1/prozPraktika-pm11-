## Описание

Приложение предназначено для ведения электронного журнала в средней общеобразовательной школе. Оно позволяет учителям и классным руководителям фиксировать оценки и посещаемость, формировать отчёты по успеваемости, а ученикам – просматривать свои данные (в перспективе при подключении авторизации).

**Ключевые возможности:**
- Полный CRUD для списка учащихся.
- Учёт оценок с типами контроля (текущая, контрольная, самостоятельная).
- Учёт посещаемости с фиксацией причин отсутствия.
- Аналитические отчёты с экспортом в CSV и Excel.
- Резервное копирование и восстановление базы данных.

## Функциональные возможности

### Ученики
- Добавление / редактирование / удаление / обновление списка учеников.
- Фильтрация по классу и ФИО.

### Оценки и посещаемость
- Выбор ученика из выпадающего списка.
- Добавление, редактирование и удаление оценок (предмет, оценка, тип контроля, дата).
- Добавление, редактирование и удаление записей посещаемости (дата, статус, примечание).
- Фильтрация оценок по предмету, посещаемости – по статусу.

### Отчёты
- **Отличники** – топ‑5 учеников с оценками только 4 и 5 (фильтр по классу).
- **Средний балл по классам** – агрегация по учебному году (2024–2025 / 2025–2026).
- **Успеваемость по предметам** – средняя оценка по предметам с фильтром по классу.
- **Отчёт по посещаемости** – для выбранного класса и диапазона дат (суммы присутствий, прогулов, больничных).

### Экспорт
- Все отчёты можно сохранить в форматы **CSV** и **Excel (.xlsx)** (библиотека EPPlus).

### Резервное копирование
- Создание полной резервной копии базы данных через SSMS или T-SQL.
- Восстановление базы данных из резервной копии.
